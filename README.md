# nullscience.ai — Website v1

Scrollytelling website for Nullscience cybersecurity advisory. Dark, cinematic design with scroll-driven animations. All copy in Brazilian Portuguese.

## Stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- Framer Motion (scroll-driven animations)
- TypeScript

## Run locally

```bash
cd nullscience-site
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Sections

1. **Hero** — cinematic intro with parallax scroll
2. **Sobre** — company overview and capabilities
3. **CrisisLab** — product showcase with simulated terminal UI
4. **Por Que Importa** — stats on Brazilian cyber threat landscape
5. **Como Funciona** — 4-step process with animated timeline
6. **Contato** — contact form + email CTA
